## ABOUT

Building systems where **intelligence meets the physical world.**

**Robotics & Embedded Systems** — Developing autonomous robots that navigate, perceive, and react in real environments. From ultrasonic obstacle avoidance to full ROS2-based navigation with RPLidar and A* pathfinding — I work close to the hardware.

**Creative Development** — On the other side, I build immersive web experiences using Three.js and GSAP. I believe software should not just work — it should feel good too.
